# RadEye

**RadEye** is a free, radiology-focused educational platform designed to support imaging interpretation and exam preparation for medical students and radiology trainees.

It brings together concise modality-based notes, real-world imaging cases, image-based MCQs, and curated resources in one structured, distraction-free space.
